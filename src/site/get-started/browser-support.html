Browser support

We aim to support the following browsers and devices:

  • Browsers
    • Chrome (latest) — Mac OSX / Windows
    • Firefox (latest) — Mac OSX / Windows
    • Safari (latest) — Mac OSX only
    • Internet Explorer (version 9+) — Windows only
  • Devices
    • TBD

We’ll review browser and device usage about every six months and adjust this list as necessary. We plan to support browsers and devices that are still used by a certain threshold of users and viewers.

Tangram requires WebGL, which is not supported in Internet Explorer 9 and 10. WebGL can also be disabled by an administrator. If a map or demo uses Tangram but WebGL is unavailable, please provide a fallback raster map (if possible), a link to an alternative map, or a helpful message explaining why the map cannot be viewed.

Testing

We will have a shared Browserstack account for testing on all supported browsers, platforms and devices.

To test Internet Explorer locally, we recommend using Microsoft’s virtual machines in your VM platform of choice (e.g. VirtualBox).
